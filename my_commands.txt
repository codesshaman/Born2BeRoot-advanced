1. SSH settings in your host

1.1. Create folder for ssh keys:

mkdir ~/.sshkeys
cd ~/.sshkeys

1.2. Generate ssh key:

ssh-keygen -t rsa -b 4096 -f id_rsa

1.3. Push publickey to server:

ssh-copy-id -i id_rsa.pub user@server_address

1.4. Add private key to your host: 

ssh-add ~/.sshkeys/id_rsa

1.5. Check the local public keys:

ssh-add -l

2. SSH settings in your webserver:

2.1. Add new user:

adduser username

2.2. Install sudo if sudo is absent:

apt install sudo

2.3. Edit sudoers file:

nano /etc/sudoers

Add follow string:

username  ALL=(ALL:ALL) ALL

2.4. Add user to root group:

usermod -aG root username
